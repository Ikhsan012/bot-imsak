# bot-imsak

## Overview
The `bot-imsak` project is a bot designed to provide accurate imsak times for the Muslim community, especially during the month of Ramadan. This bot takes into account various geographical locations to deliver precise timings.

## Features
- Provides imsak and iftar timings based on user location.
- Supports multiple locations and time zones.
- User-friendly interface for easy interaction.
- Notifications for imsak and iftar times.

## Installation
To install the bot, follow these steps:
1. Clone the repository:  
   `git clone https://github.com/Ikhsan012/bot-imsak.git`
2. Navigate to the project directory:  
   `cd bot-imsak`
3. Install the dependencies:  
   `npm install`

## Usage
After installation, you can start the bot with the following command:
`node bot.js`

## Configuration
Set your geographical location in the configuration file to get accurate timings.
